# Shion🍂

> **Warning**
> 处于开发阶段，谨慎使用🚧



## 介绍

> 曾经读过的一本书，弹过的一首曲子。也许在某个午后，翻开它你依然能回忆起那股气息

一款时间追踪软件，帮助您记录下一段时间

+ 📊数据展示卡片
+ 📅每日时间线记录
+ 🏷️记录分类
+ 👀监听程序活动
+ 📖日记
